What was wrong with Redcat?

When Redcat was created it fulfilled the roles it was created for. However that was 5 years ago, I use my computers differently. I've Lost interest in some hobbies and gained interest in others.

Redcat had two goals.

  1. Give me a place to learn about and enjoy Linux.
  2. Store files for siblings and myself. Mainly to watch pirated movies and tv shows.

I've enjoyed linux so much that all my computers run it now. I've lost interest in watching movies and TV shows.

Redcat was running a poorly configured Ubuntu server that often required fixing. Redcat also had plenty of redundant storage filled up with useless junk.

Too Much Effort For Little Gain.
